# 01238_Differential_geometry_DG503
A library for Differential Geometry computations.

## Project Structure
- `src/functions/`: Core mathematical functions.
    - `functions.py`: General differential geometry functions (Jacobi, Lie derivatives, Christoffel, etc.).
    - `curves.py`: Specific functions for curve analysis.
    - `surfaces.py`: Specific functions for surface analysis.
- `examples.ipynb`: Jupyter notebook demonstrating the library usage.

## Getting Started
To use these functions in a script or notebook:
```python
from functions import calculate_jacobi_function
```
